# Dasonic Website (Starter Repo)

This is a clean **Next.js + TypeScript + Tailwind** starter for the Dasonic company website.

## What’s inside
- Next.js App Router (`/app`)
- TailwindCSS styling
- Basic pages: Home, About, Services, Projects, Contact
- Company profile images copied to: `public/company-profile/`

## Run locally

```bash
# 1) install deps
npm install

# 2) start dev server
npm run dev
```

Then open `http://localhost:3000`.

## Create the git repo (local)

```bash
git init
git add .
git commit -m "chore: initial website scaffold"
```

## Push to GitHub

```bash
# replace with your repo url
git remote add origin https://github.com/<your-username>/dasonic-website.git
git branch -M main
git push -u origin main
```
